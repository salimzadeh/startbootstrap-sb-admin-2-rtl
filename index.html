رویکرد توسعه

SB Admin 2 به منظور کاهش استفاده از CSS و رفع عملکرد ضعیف صفحه، از کلاس های آماده فریم‌ورک Bootstrap 4 استفاده گسترده می کند. کلاسهای CSS سفارشی، برای ایجاد اجزا و ابزارهای سفارشی استفاده می شود.

پیش از شروع کار با این قالب، شما باید با چارچوب های فریم‌ورک Bootstrap، به ویژه کلاس های کاربردی آن، آشنا شوید.
